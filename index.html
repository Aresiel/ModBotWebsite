What is Clone Drone Mod-Bot?

It's a mod loader and manager for Clone Drone in the Danger Zone, which lets modders add lots of cool new stuff to the game.

Note that you need to have bought the game Clone Drone in the Danger Zone to use Mod-Bot with it.

How do I get started?

If you have bought Clone Drone in the Danger Zone, all you need to do is download the Mod-Bot application and follow the installation steps. Then you can choose which mods to download, either by using the in-game UI or by visiting the mod browsing page.

How do I make stuff?

For guide on how to create your own custom levels, please visit the level editing page. Otherwise if you want to make mods, please visit the modding page. Note that some previous programming knowledge is needed.

You will need a verified account to upload mods or tags. To get your acount verified, simply log in with it in-game to link it with your steam account.

If you have any questions or suggestions, please share them in the Clone Drone Mod-Bot Discord Server.
